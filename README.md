# DecodeLabs Data Analytics Internship – Project 1

## Data Cleaning & Preparation

This project was completed as part of the DecodeLabs Data Analytics Internship.

### Objective

Clean a raw e-commerce dataset by identifying and correcting data quality issues to prepare the data for analysis.

### Tools Used

* Python
* Pandas
* Visual Studio Code
* Microsoft Excel

### Data Cleaning Tasks Performed

* Identified missing values
* Handled missing CouponCode entries
* Checked duplicate records
* Verified unique Order IDs
* Converted date columns to datetime format
* Standardized text values
* Validated numeric columns
* Performed final data quality checks

### Dataset Summary

* Records: 1,200
* Columns: 14
* Missing CouponCode values handled: 309
* Duplicate records found: 0
* Clean dataset generated successfully

### Files Included

* Original Dataset
* Cleaned Dataset
* Python Script
* Project Presentation
* Project Screenshots

### Learning Outcomes

This project helped me understand the importance of data cleaning and preparation before performing any analysis or visualization. It also provided hands-on experience using both Python (Pandas) and Excel for real-world data quality improvement tasks.
